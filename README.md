# Daniel Sisson Personal Site

Minimal futuristic one-page personal website built with plain HTML, CSS, and JavaScript.

## Stack

- HTML
- CSS
- JavaScript
- Fully static and GitHub Pages friendly

## Structure

- `index.html`
- `styles.css`
- `script.js`
- `CNAME`
- `assets/me.png`

## Local Preview

Open `index.html` directly in a browser, or serve the folder with any static file server.

## GitHub Pages

This site is fully static and can be deployed directly with GitHub Pages. The included `CNAME`
file is configured for:

`danielsisson.com`
